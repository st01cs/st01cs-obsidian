在做视频剪辑的时候，素材库里存在几百个视频文件，想从中找到某个就镜头，需要耗费大量时间。  
  
无独有偶，在 GitHub 上发现 Edit Mind 这款开源的桌面应用，利用 AI 为我们的视频库建立了一个 “智能大脑”。  
  
首先，它对本地视频进行深度分析，自动提取完整转录文本、识别人脸、检测物体、提取屏幕文字甚至分析主色调。  
  
然后，我们就可以用自然语言搜索，轻松找到某个视频片段，甚至还可以让它帮我们直接剪辑生成视频。  
  
GitHub：github.com/IliasHad/edit-mind  
  
所有分析完全可以在本地执行，视频文件不会上传云端，不过对设备的性能配置要求比较高。  
  
目前项目还在开发中，功能不太完善，想要提前尝鲜的开发者，克隆代码到本地安装依赖，简单配置下即可运行。
