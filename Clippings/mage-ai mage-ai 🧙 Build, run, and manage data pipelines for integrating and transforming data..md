### Quickstart

Install using Docker (recommended):

```
docker pull mageai/mageai:latest
```

Or with pip:

```
pip install mage-ai
```

Or with conda:

```
conda install -c conda-forge mage-ai
```

Full setup guide and docs: [docs.mage.ai](https://docs.mage.ai/)
