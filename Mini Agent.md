Mini Agent 是一个极简但功能强大的演示项目，展示了如何使用 MiniMax M2 模型打造高效智能 Agent。它兼容 Anthropic API，支持交错思维，能够处理复杂且长时间的推理任务。  
  
项目特点包括：  
  
- 完整的 Agent 执行循环，集成文件系统和 Shell 操作工具；  
- 持久化记忆功能，通过 Session Note Tool 记录多轮会话信息；  
- 智能上下文管理，自动摘要会话历史，支持超长上下文；  
- 内置 15 种 Claude Skills，涵盖文档处理、设计、测试等领域；  
- 支持 MCP 协议，方便接入知识图谱和网页搜索工具；  
- 详细日志记录，便于调试和追踪执行过程；  
- 简洁美观的命令行界面，代码结构清晰，易于扩展和二次开发。  
  
支持快速上手和开发两种模式，无论是新手体验还是专业开发都非常友好。项目代码开源，文档完善，适合想探索智能 Agent 构建的开发者和研究者使用。
