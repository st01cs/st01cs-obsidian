平时研究某只股票时，需要在各种网站之间来回切换，查看股价、财报、新闻等信息，想要快速了解一家公司的基本面和最新动态，确实挺耗时。  
  
可以看一下， AI Financial Agent 这个开源项目，它将复杂的投资研究变成了一场与 AI 的简单对话，颇为强大。  
  
基于 Financial Datasets API 构建，覆盖美股 30 年历史数据，包括财务报表、股价、期权、内部交易等信息，并以图表方式直观的展示股票价格和基本面数据。  
  
GitHub：github.com/virattt/ai-financial-agent  
  
主要功能：  
  
- 对话式股票分析，通过聊天方式获取投资研究信息；  
- 实时和历史股票数据查询，覆盖美股 100% 市场；  
- 可查询财务报表、期权数据、内部交易等多维度信息；  
- 生成式 UI 展示，让复杂的财务数据更易理解；  
- 基于 LangChain 和 OpenAI 构建，支持自定义扩展。
