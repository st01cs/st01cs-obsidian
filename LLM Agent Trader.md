做股票交易策略回测，总是需要结合大量数据和复杂模型，单靠手工分析效率低且容易出错。  
  
LLM Agent Trader 是一个基于大语言模型的智能交易回测系统，集成了AI辅助的决策分析，帮助用户更高效地设计和验证交易策略。  
  
它支持多种环境，提供灵活的配置和一键启动方式，方便在本地快速搭建使用。通过调用不同的LLM服务（如Azure OpenAI或Google Gemini），实现智能交易信号的生成和分析。  
  
主要特点包括：  
  
- 集成大语言模型，提升交易策略分析的智能化水平；  
- 支持macOS、Linux原生环境，Windows用户可通过WSL或Git Bash使用；  
- 简单的依赖安装与环境配置流程，快速启动系统；  
- 提供命令行工具管理服务的启动、停止、测试和代码格式化；  
- 灵活切换不同的语言模型API，适应多种AI服务需求。
